JS loader plugin

This is a simple JS module to add a loading overlay to one or more Elements.

basic usage

import Loader from './loader';

const ldr = new Loader({
    image: './img/loding.gif'
});

document.querySelector('.yourElement').addEventListener('click', (e) => {
    e.preventDefault();
    ldr.show();
})

You can pass an options object to the plugin with several parameters. At the momente these parameters are:

  • image - (url of an image)
  • backgroundClass - (class of the background element)
  • loaderClass - (the class name of the loader element)

You can also pass a options object to the show and hide function that can contain the following values

  • elements - a jQuery object (or a array of objects) that will be overlayed by the loader
  • after - a function that will be executet right after the function finishes

Example

import Loader from './loader';

const ldr = new Loader({
	image: './img/loding.gif'
});

document.querySelector('.yourElement').addEventListener('click', (e) => {
    e.preventDefault();
    ldr.show({
        after: function() {
            window.setTimeout(function () {
                ldr.hide({
                    after: () => {
                        alert('loading finished');
                    }
                });
            }, 3000);
        }
    });
})
